# admr-fc-vs-vo-misspecification

# Comparison of Full Covariance and Variance-Only Aggregate Data Modeling under PK Model Misspecification

This repository contains code for a simulation study comparing two aggregate data modeling strategies using the deprecated `admr` R package:

1. **Full covariance approach (FC)**: uses the full variance-covariance matrix of aggregate observations.
2. **Variance-only approach (VO)**: uses only the diagonal variance terms and ignores covariance between time points.

The aim is to evaluate how these two approaches perform when the population pharmacokinetic model is misspecified by removing one inter-individual variability term at a time.

## Study objective

Aggregate data modeling is useful when individual-level pharmacokinetic data are unavailable and only summary statistics are reported. However, published aggregate data may contain either full covariance information or only variances/standard deviations.

This study investigates whether using the full variance-covariance matrix improves robustness compared with using variance-only data under random-effect misspecification.

## Model

The base model is a two-compartment pharmacokinetic model with first-order absorption:

- Clearance: `cl`
- Central volume: `v1`
- Peripheral volume: `v2`
- Inter-compartmental clearance: `q`
- Absorption rate constant: `ka`

The true model includes inter-individual variability on all five parameters. Misspecified models are generated by removing IIV on one parameter at a time:

- remove IIV on `cl`
- remove IIV on `v1`
- remove IIV on `v2`
- remove IIV on `q`
- remove IIV on `ka`

## Methods

The analysis uses the `admr` package and its Monte Carlo fitting algorithm, `fitMC()`.

For each misspecification scenario, the model is fitted twice:

- once using the full covariance matrix
- once using variance-only data

Model performance is evaluated using:

- parameter relative bias
- BSV relative bias
- simulated concentration-time profiles
- comparison of prediction intervals against the true model
